# CLAUDE.md

## About this project

This is a React website built with Vite for Empower Recovery, an addiction medicine practice.

The goal is to create a modern, professional, responsive, accessible healthcare website.

## About me

I am completely new to programming and React.

Assume I don't know programming terminology unless you've explained it already.

Teach me as you build.

## How I want you to work

Before making any code changes:

- Explain what you're going to do.
- Explain why you're doing it.
- Tell me which files you'll modify.

After making changes:

- Explain what changed.
- Explain every new React concept you introduced.
- Explain any JavaScript that might be unfamiliar.

## Coding rules

- Keep code simple and beginner-friendly.
- Prefer readability over cleverness.
- Use functional React components.
- Keep components small and focused.
- Use descriptive variable names.
- Comment only when it helps explain a concept.
- Do not duplicate code unnecessarily.

## Safety rules

Never:

- Install a package without asking me first.
- Delete files unless I approve it.
- Rename folders without asking.
- Make large project-wide changes in one step.

Instead:

- Build one feature at a time.
- Wait for my approval before moving to the next feature.

## UI guidelines

The website should feel:

- Modern
- Professional
- Calm
- Trustworthy
- Premium
- Clean
- Mobile-first
- Accessible

Avoid flashy animations.

Prioritize readability and fast loading.

## Design preferences

Use:

- Plenty of white space
- Soft colors
- Rounded corners
- Professional typography
- Clear call-to-action buttons
- Responsive layouts

## When I ask for a feature

Always:

1. Explain the plan.
2. Build it.
3. Explain the code.
4. Suggest improvements.
5. Wait for my approval before continuing.

## If there is an error

Don't just fix it.

Explain:

- Why it happened.
- How you found it.
- How to avoid it in the future.

## Goal

Help me become confident enough to build this website myself instead of just copying code.
